# μPD777エミュレータ

# これは何？
μPD777というののエミュレータです。

# ビルド環境
* Windowsのみ
* C++20
* CMake

## ビルド方法
1. rom.cppにROMイメージ、パターンを書きます
2. CMakeにてプロジェクトを作成し
3. ビルドします
