# μPD777エミュレータ

# これは何？
μPD777というののエミュレータです。  

# ビルド環境
* Windowsのみ
* C++20
* CMake

## ビルド方法
1. CMakeにてプロジェクトを作成し
2. ビルドします

# コマンドラインオプション

| オプション | 説明 |
|---|---|
| -code <コードファイル名> | コードファイルを読み込む |
| -ptn <パターンファイル名> | パターンファイルを読み込む |
| -disassemble | コードを逆アセンブルし終了 |

ファイルのフォーマットなどは ⇒ [PD777supplement Pages](https://github.com/W88DodPECuThLOl/PD777supplement)
